# KeyboardTools
Simple AHK script that create shortcut for cool symbols

## Shortcut

``Alt`` + ``-`` for get a `—` 

``Shift`` + ``Alt`` + ``<`` for get a `«` 

``Shift`` + ``Alt`` + ``>`` for get a `»`
